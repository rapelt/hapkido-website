Hapkido Brisbane

Integrity • Respect • Discipline

At Hapkido Brisbane we believe being able to defend yourself is a right not a privilege. We are dedicated to teaching self defence to anyone willing to learn and are commited to providing a fun and safe environment for everyone to learn.

Lifelong values

Integrity, respect and discipline are the core values all Hapkido Students learn and live by. It is how we train safely together and how we continue to create a friendly environment for everyone to train together.

Get with with fun

Every class provides plenty of exercise for every fitness level. Each class covers a basic warm up, kicking, punching, techniques, breakfalls and lot more. Class is never boring or monotonous when learning Hapkido.

Practical skills

Hapkido is a martial arts that is focused on practical self defence skills. Students learn useful skills from the first class to be able to defend themselves from punches, kicks, grabs and lots more.
